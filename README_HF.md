# 🔍 Deepfake Detection AI

Aplikasi deteksi deepfake menggunakan Deep Learning dengan arsitektur **ResNet-50**.

## 🎯 Fitur

- ✅ Upload gambar wajah
- ✅ Deteksi Real vs Fake
- ✅ Confidence score
- ✅ Penjelasan hasil deteksi
- ✅ Interface yang user-friendly

## 🧠 Model

- **Architecture**: ResNet-50
- **Training**: 5-Fold Cross-Validation
- **Dataset**: 1,722 gambar (Real + Fake)
- **Accuracy**: ~85-90%

## 🚀 Cara Pakai

1. Upload gambar wajah
2. Klik "Detect Deepfake"
3. Lihat hasil deteksi dan confidence score

## 📝 Disclaimer

Hasil deteksi tidak 100% akurat. Gunakan sebagai referensi tambahan, bukan satu-satunya sumber verifikasi.
